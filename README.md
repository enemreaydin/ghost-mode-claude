# 👻 Ghost Skill for Claude

**Make Claude write like a human. Bypass AI detectors. Leave no trace.**

Ghost is a skill for Claude that transforms its output style — making it sound natural, imperfect, and human. It mimics real writing patterns: variable sentence length, subtle grammar flaws, structural inconsistencies, and non-native phrasing. All configurable.

Works with both **Claude Code** and **Claude.ai** (web).

---

## ✨ What It Does

When `/ghost` is active, Claude:

- Writes with **irregular sentence and paragraph lengths** instead of perfect AI structure
- **Avoids AI-typical connectors** like *"Firstly, moreover, in conclusion"*
- Introduces **subtle, natural imperfections** — not random errors, but realistic ones
- Can mimic a **non-native speaker's phrasing** to further reduce detection signals
- Can produce **minor grammar/punctuation errors** that only a professional would catch
- **Overexplains or underexplains** topics the way a real human would

## 📦 Installation

### Mac / Linux

```bash
git clone https://github.com/enemreaydin/ghost-mode-claude.git
cd ghost-mode-claude
chmod +x install.sh
./install.sh
```

Or with one line:

```bash
curl -fsSL https://raw.githubusercontent.com/enemreaydin/ghost-mode-claude/main/install.sh | bash
```

### Windows (PowerShell)

```powershell
git clone https://github.com/enemreaydin/ghost-mode-claude.git
cd ghost-mode-claude
.\install.ps1
```

Or with one line:

```powershell
irm https://raw.githubusercontent.com/enemreaydin/ghost-mode-claude/main/install.ps1 | iex
```

## ⚙️ Setup

The installer will ask you 4 questions:

| # | Question | What it does |
|---|---|---|
| 1 | **Primary language** | Sets the native language context for Claude |
| 2 | **Non-native speaker mode** | Claude uses phrases typical of non-native speakers |
| 3 | **Intentional grammar errors** | Subtle typos and punctuation mistakes |
| 4 | **Structural imperfections** | Human-like flow: uneven focus, mild redundancy |

After answering, a summary is shown and you confirm before anything is installed.

## 🚀 Usage

### Claude Code

```
/ghost
```

Type this in any Claude Code session to activate ghost mode.

### Claude.ai (Web)

After running the installer, a `ghost-skill.zip` file is created at your home directory (`~/ghost-skill.zip` on Mac/Linux, `%USERPROFILE%\ghost-skill.zip` on Windows).

Upload it manually:

1. Go to [claude.ai](https://claude.ai)
2. Open **Settings → Features → Skills**
3. Upload `ghost-skill.zip`
4. Start a new chat — ghost mode is now available

## ⚠️ Notes

- Claude.ai requires **code execution to be enabled** in settings for skills to work
- Ghost mode affects **writing style only** — Claude's knowledge and capabilities remain the same
- If Claude Code is not installed, the installer skips that step and only creates the zip
